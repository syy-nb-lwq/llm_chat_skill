# Heartbeat - 主动任务清单

配置需要定期执行的主动任务。Agent 会自动检查并执行这些任务。

## 任务格式

每个任务以 `- ` 开头,支持以下触发条件:

- `每天 HH:MM` - 每天指定时间执行
- `每隔 N 分钟` - 固定间隔执行
- `每隔 N 小时` - 固定间隔执行
- `每周X HH:MM` - 每周指定星期和时间执行

## 任务列表

# 日常工作
- 每天 8:00 发送今日日程摘要
- 每隔 30 分钟检查一次重要邮件

# 项目管理
- 每隔 1 小时检查 CI 构建状态
- 每天 18:00 发送工作进度报告

# 提醒事项
- 每天 9:30 提醒站立会议
- 每隔 2 小时提醒休息喝水

# 健康提醒
- 每隔 45 分钟提醒站起来活动
- 每天 12:00 提醒午餐时间
