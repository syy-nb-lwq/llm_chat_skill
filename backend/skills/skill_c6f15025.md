# 技能：旅游行程规划

## 能力
根据出发地、目的地、时间和偏好，制定详细的旅游行程，包括交通、景点、住宿、餐饮等建议

## 匹配模式
- 想要去某地旅游
- 帮忙安排行程
- 周末去哪里玩

## 方法论
分析用户需求，结合地理距离、交通方式、景点开放时间、天气等因素，生成合理的时间安排

## 步骤
1. 1. 确定出发地和目的地
2. 2. 查询两地之间的交通方式及时间
3. 3. 了解目的地主要景点及开放时间
4. 4. 根据停留时间分配景点顺序
5. 5. 建议住宿和餐饮
6. 6. 输出完整行程表
